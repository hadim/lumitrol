# Lumitrol

Control your Lumix camera from your phone or tablet. View and download images and **videos**.

Tested camera:

- DC-GX9

## Installation

On Google Play Store (link to come).

## Contribute

- Download and install [Android Studio](https://developer.android.com/studio).
- Hack the code.
- Submit a PR.
